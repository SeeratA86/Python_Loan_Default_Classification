# Python Loan Default Classification

A data analytics and machine learning project focused on predicting loan default outcomes using Python and Jupyter Notebook.

This project applies data preprocessing, exploratory analysis, model training, model comparison, and prediction generation using multiple classification algorithms.

## Project Overview

This project aimed to build classification models that predict whether a loan applicant is likely to default.

The target variable is loan default, represented as:

- `0` = No loan default
- `1` = Loan default

The project follows a practical data science workflow where several classifiers are trained, tested, compared, and used to generate prediction outputs for an unseen dataset.

## Classification Models Used

The project explores multiple machine learning classifiers:

- Decision Tree
- K-Nearest Neighbour
- Random Forest
- Support Vector Machine
- Neural Network

These models were compared to identify which classifier produced the most reliable results for the loan default prediction task.

## Project Workflow

The notebook follows these main steps:

1. Load and inspect the dataset
2. Explore the dataset and understand the available features
3. Preprocess and transform the data where required
4. Split the data into training and testing sets
5. Train multiple classification models
6. Evaluate model performance
7. Compare classifier results
8. Generate prediction CSV files for the unseen dataset
